# Bike Sales Data Analysis

This repository contains a project that analyzes bike sales data using Excel and presents the insights through an interactive dashboard. The dashboard showcases various visualizations that help understand the relationship between different factors and bike purchase decisions. The dataset used for this analysis contains information about individuals' characteristics and whether they purchased a bike or not.

## Project Overview

The goal of this project was to analyze bike sales data to uncover patterns and trends related to bike purchases. The dataset includes the following attributes:

- ID
- Marital Status
- Gender
- Income
- Children
- Education
- Occupation
- Home Owner
- Cars
- Commute Distance
- Region
- Age
- Age Group
- Purchased Bike

## Dashboard Features

The interactive dashboard provides visualizations that facilitate the exploration of relationships between various attributes and the decision to purchase a bike. The dashboard includes the following sections:

### Gender vs. Income
- Displays a comparison of the gender-wise distribution of income among individuals who purchased a bike and those who did not.

### Age Group vs. Purchase Decision
- Presents a breakdown of age groups and their corresponding purchase decisions, helping identify any trends across different age categories.

### Commute Distance vs. Purchase Decision
- Illustrates the relationship between the distance individuals commute and their decision to purchase a bike.

### Slicers for Additional Insights
The dashboard is enhanced with slicers that allow users to filter the data based on marital status, education, and region. This enables more in-depth exploration of the data and the ability to uncover hidden insights.

## How to Use the Dashboard

1. **Download the Excel File:** Clone or download this repository to your local machine.

2. **Open the Excel File:** Open the Excel file named `Bike Sales Dashboard.xlsx` using Microsoft Excel or any compatible spreadsheet software.

3. **Interact with the Dashboard:** Navigate through the different visualizations to explore the relationships and patterns in the data. Use the slicers to filter the data based on marital status, education, and region for more specific insights.

## Dataset Source

The dataset used for this analysis is included within the Excel file itself. It comprises a sample of individuals' characteristics and their bike purchase decisions.
